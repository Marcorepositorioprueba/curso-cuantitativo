Clase 3: Conectando los Puntos - Análisis Descriptivo de Datos Bivariados

Descubre cómo dos variables se relacionan entre sí y aprende a medir la fuerza y dirección de su asociación para generar evidencia más robusta.

1. Introducción al Análisis Bivariado

Hasta ahora, hemos analizado variables de una en una (análisis univariado). Pero la realidad es compleja y las cosas rara vez ocurren de forma aislada. El verdadero poder del análisis de datos surge cuando empezamos a conectar los puntos. Eso es el análisis bivariado: el estudio de dos variables de forma simultánea para entender su relación.

Este análisis nos permite responder preguntas como:

  • ¿Afecta la inversión en educación (variable A) a las tasas de criminalidad (variable B)?
  • ¿Hay alguna relación entre el género de una persona (variable A) y su salario (variable B)?
  • ¿El nivel de satisfacción laboral (variable A) depende del departamento en que trabaja una persona (variable B)?

En esta clase, exploraremos los tres escenarios principales del análisis bivariado:

  1. Numérica vs. Numérica: Correlación y diagramas de dispersión.
  2. Categórica vs. Categórica: Tablas de contingencia.
  3. Numérica vs. Categórica: Comparación de grupos y gráficos de cajas.

2. Numérica vs. Numérica: Correlación

Este es el escenario más común. Queremos saber si a medida que una variable numérica sube, la otra sube, baja o no hace nada. La herramienta visual por excelencia es el diagrama de dispersión, que ya vimos en la clase anterior.

Para cuantificar esta relación, usamos el coeficiente de correlación de Pearson (r). Este número nos dice dos cosas:

  • Dirección: Si es positivo, ambas variables tienden a subir juntas. Si es negativo, cuando una sube, la otra tiende a bajar.
  • Fuerza: El valor va de -1 a 1. Cerca de -1 o 1 indica una relación lineal muy fuerte. Cerca de 0 indica una relación lineal débil o nula.
Ejemplo: Analizando la relación entre años de experiencia y salario.

Observamos que a más años de experiencia, generalmente hay un mayor salario. El coeficiente de correlación de r = 0.85 confirma una fuerte relación positiva.

¡Correlación no implica Causalidad!

Este es el error más famoso en estadística. Que dos variables se muevan juntas no significa que una cause a la otra. Por ejemplo, las ventas de helado y los ataques de tiburones están correlacionadas (ambas aumentan en verano), pero comer helado no provoca ataques de tiburón. La causa real es una tercera variable: el calor del verano.

3. Categórica vs. Categórica: Tablas de Contingencia

Cuando trabajamos con dos variables categóricas (ej. 'género' y 'partido político'), usamos tablas de contingencia o tablas cruzadas. Estas tablas nos muestran cuántas observaciones caen en cada combinación de categorías.

Ejemplo: ¿La preferencia por el trabajo (remoto vs. oficina) depende del departamento?

4. Numérica vs. Categórica: Comparación de Grupos

¿Cómo se comporta una variable numérica (ej. 'salario') en diferentes grupos (ej. 'ciudades')? Aquí, calculamos estadísticas descriptivas (media, mediana, desviación estándar) para la variable numérica en cada categoría de la variable categórica.

Una de las mejores herramientas visuales para esto es el gráfico de cajas (Box Plot). Un box plot nos muestra:

  • La mediana (la línea dentro de la caja).
  • El rango intercuartílico (la caja en sí, que contiene el 50% central de los datos).
  • Los valores mínimos y máximos ("los bigotes").
  • Los valores atípicos (puntos fuera de los bigotes).
Ejemplo: Comparar la calificación de desempeño (numérica) de empleados en tres turnos diferentes (categórica).

Conclusión: Visualmente, es inmediato que el turno de la mañana tiende a tener calificaciones más altas y más consistentes (caja más pequeña) que el turno de la noche.

Profundizando: Varianza, Covarianza y Correlación

Para entender matemáticamente cómo se relacionan dos variables, necesitamos tres conceptos clave. Usaremos una pequeña muestra de nuestros datos de la encuesta para ilustrarlo.

Paso 1: Varianza (Repaso)

La varianza (σ²) nos dice qué tan dispersos están los datos de una variable respecto a su media. Una varianza alta significa datos muy dispersos; una baja, datos muy juntos.

Fórmula (para una muestra): σ² = Σ(xᵢ - μ)² / (n-1)

Paso 2: Covarianza

La covarianza nos dice si dos variables se mueven en la misma dirección (positiva) o en direcciones opuestas (negativa).

  • Covarianza > 0: Cuando X sube, Y tiende a subir.
  • Covarianza < 0: Cuando X sube, Y tiende a bajar.

El problema: El valor (ej. 192250) no nos dice la fuerza de la relación, solo la dirección. Es difícil de interpretar.

Fórmula (para una muestra): Cov(X,Y) = Σ((xᵢ - μₓ)(yᵢ - μᵧ)) / (n-1)

Paso 3: Correlación (La Solución)

El coeficiente de correlación (r) es simplemente la covarianza estandarizada. Se divide la covarianza por el producto de las desviaciones estándar de ambas variables. Esto ajusta el valor a un rango fijo de -1 a 1, haciéndolo fácil de interpretar.

Fórmula: r = Cov(X,Y) / (σₓ * σᵧ)

Ejemplo de Cálculo: Correlación entre Edad (X) e Ingreso (Y)

Usaremos los primeros 4 registros de encuesta_socioeconomica.csv:

  • Edades (X): {25, 45, 33, 58}
  • Ingresos (Y): {15000, 35000, 22000, 45000}
  1. Calcular Medias:
    • Media de Edad (μₓ) = (25 + 45 + 33 + 58) / 4 = 40.25
    • Media de Ingreso (μᵧ) = (15k + 35k + 22k + 45k) / 4 = 29,250
  2. Calcular Covarianza:
    • Sumatoria de productos de desviaciones: ((25-40.25)*(15k-29.25k)) + ... = 576,750
    • Cov(X,Y) = 576,750 / (4-1) = 192,250 (Positiva, como esperábamos)
  3. Calcular Varianzas y Desviaciones Estándar:
    • Varianza de Edad (σ²ₓ) = 207.58 => Desv. Est. (σₓ) = 14.41
    • Varianza de Ingreso (σ²ᵧ) = 178,916,667 => Desv. Est. (σᵧ) = 13,376
  4. Calcular Correlación:
    • r = 192,250 / (14.41 * 13,376) = 0.997

Conclusión: Obtenemos un coeficiente de correlación de r ≈ 0.997, lo que indica una relación lineal positiva casi perfecta para esta pequeña muestra.
